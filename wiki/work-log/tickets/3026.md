## Body

## Epic: Zero-Miss Self-Annealing Guardrails

### Problem

The harness self-annealing system (`workflow-resilience.instructions.md`) is advisory text enforced by operator judgment. Three compounding gaps cause governance drift to silently escape without a ticket:

1. **No recurrence gate on anneal decisions**: The operator can classify a flaw as `memory-note-only` even when the same `pattern_id` already exists in `incidents.jsonl` — proving memory alone is insufficient. No validator rejects this.
2. **No automated Tier-2 promotion**: `incidents.jsonl` recurrence data exists but nothing reads it to auto-file a ticket when a pattern_id crosses the recurrence threshold (≥2 in 7d).
3. **No session-boundary check**: A session can end with an open PR whose linked issue has no CONSULTANT_CLOSEOUT, with no warning emitted.

### Goal

100% adherence: every governance drift event that meets Tier-2 criteria produces a ticket automatically, without relying on operator judgment to classify correctly.

### Acceptance Criteria (Phase 1)

- **C1** (#3025): Validator rejects `memory-note-only` anneal decision when same `pattern_id` already present in `incidents.jsonl`. Pre-session-end advisory check emits warning when open PRs have linked issues without CONSULTANT_CLOSEOUT.
- **C2** (new): Automated Tier-2 ticket promotion — a script/hook reads `incidents.jsonl`, groups by `pattern_id`, and files a Manager ticket to backlog when recurrence ≥ 2 within 7 days and no open ticket for that pattern already exists. Idempotent (no duplicate tickets).
- **C3** (new): Pre-push hook addition to detect open PRs with linked issues missing CONSULTANT_CLOSEOUT and block push with a recoverable error (not a hard deny — emit warning + allow with explicit override marker `[skip-closeout-check]`).

### Out of Scope (Phase 2)

- Fully automated baton completion (requires agent autonomy beyond current harness model)
- Ticket auto-close on pattern resolution

### Definition of Done

- All three children closed
- `npm test` passes with new validators under test
- 0 incidents escape to manual-pull without a ticket for ≥ 30 days post-ship
