# #2678 — Enforce Phase-0 to Phase-1 promotion gate across all AI runtimes (research-first)

> **Source**: github:issue/2678 | **state: OPEN** | **Labels**: type:epic, priority:P1, role:manager, area:governance, lane:code-change, status:dormant, phase-gate:research-first
> Mirror of `gh issue view 2678` (derived; edit the GitHub item, not this page).

## Body

## Objective

Phase-0 research-first Epics complete their research children and close terminally without triggering automatic Phase-1 child-ticket creation. The entire Phase-0 to Phase-1 promotion path is instruction-only guidance with no hard automation. This Epic ships the enforcement layer.

## Problem

On 2026-06-05, Epic #2661 closed terminally after Phase-0 child #2662 completed with no Phase-1 development children created. Root-cause analysis confirmed:

- research-first-phase-gate.js: advisory-only; validates EPIC_RESCOPE comment only; no Phase-1 child predicate
- epic-ac-traceability.js: zero-AC-R escape hatch silently passes Epics with no AC-R markers
- research-first-phase-gate-lint.yml: explicit core.warning() not core.setFailed(); does not block Epic closure
- baton-gates.yml: skips lane:docs-research entirely; Phase-0 lane bypasses all artifact checks
- closeout-lint.yml: no Phase-1 child check
- label-lint-close-protection.js: allows close if CONSULTANT_CLOSEOUT present; no phase check
- Historical scan: ~50% of 36 research-first Epics have Phase-1 children; all created by manual agent instruction-following, never automated

Secondary cause: Advisory-only enforcement makes outcomes model-capability-dependent. Frontier models self-enforce instructional governance; weaker Auto-routed models can skip Phase-1 child creation with no automated safety net.

## Scope

1. phase0_green_complete(epicNumber) predicate -- detects all Phase-0 children closed with consultant evidence plus Epic has EPIC_RESCOPE comment
2. Auto-materialize Phase-1 child-ticket template bundle when predicate fires
3. Closure-block guard on Epic if predicate is true but no Phase-1 children exist
4. Cross-runtime parity (Copilot / Codex / Claude Code / Antigravity)
5. Observability events emitted to ~/.megingjord/incidents.jsonl
6. Regression tests preventing reversion to advisory-only behavior
7. Documentation contract update (instructions/epic-governance.instructions.md + skills)
