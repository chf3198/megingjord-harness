## Body

## Trigger

Tier-2 anneal: `lint-required` CI job ("Pre-push parity smoke" step) exits 1 in GitHub Actions while the identical command (`node scripts/global/pre-push-gates.js` → `npx lefthook run pre-push`) exits 0 locally on the same branch. Blocks #1990 (#1971 PR) merge and likely blocks every subsequent PR that touches scripts/global/.

## Reproduction

- Branch: `feat/1971-cyclomatic-complexity`
- Local: `node scripts/global/pre-push-gates.js` → exit 0, all 11 lefthook steps ✔️
- CI: same command → exit 1, only "Pre-push parity smoke" step name appears in log; later steps never run
- ESLint reports 1357 baseline warnings PROJECT-WIDE (lines 59:1 and 67:1 reference `body`/`provenance`/`options` params in OTHER files, not the new complexity-report.js)
- My new file `scripts/global/complexity-report.js` lints standalone to exit 0
- `lint:js` has `--max-warnings 9999`; baseline 1357 < 9999 should pass
- `lefthook.yml` has `npm run lint:js || true` so lint:js failure cannot propagate

## Observed pattern

- PR #1986 (Copilot Team C3a) merged with the same CI gates — suggests gate is not deterministically broken
- PR #1990 (mine) failing on the SAME command Copilot's PR passed
- Difference: #1990 adds a NEW file in `scripts/global/`; #1986 adds files in `instructions/` and `wiki/concepts/`

## Hypotheses

1. **Lefthook parallel:true race** — In CI, parallel step ordering may cause one step to consume stdout/stderr from another in a way that makes the wrapper read non-zero exit
2. **`|| true` shell expansion** — CI may invoke commands via a shell that doesn't expand `|| true` the same way (POSIX vs bash difference)
3. **eslint-plugin-jsdoc fresh install** — CI's `npm install --no-audit --no-fund` produces a fresh node_modules with possibly newer plugin version that reports more warnings
4. **node 22 vs local** — CI uses node:22; local may differ

## Acceptance criteria

- [ ] AC3: fix at the root cause (whichever of the 4 hypotheses pans out)
- [ ] AC4: regression fixture in `tests/ci-pre-push-parity.spec.js` that fails if local diverges from CI on this command set
- [ ] AC5: PR #1990 (#1971) CI green and merged after fix lands
