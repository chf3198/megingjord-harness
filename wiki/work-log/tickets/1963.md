## Goal

Produce a structured gap analysis of the 2026 AgentOps/agentic governance landscape against the current Megingjord harness enforcement primitives (mapped in `wiki/concepts/harness-goal-controls.md`). Output is consumed by Child R2 to draft the Phase-1 implementation plan.

## Scope (what to survey)

Each item below must produce a written finding under `research/goal-enforcement-2026-05-19.md` (or equivalent dated research file):

1. Microsoft Agent Governance Toolkit (MIT, April 2026) — components: Agent OS, Agent Mesh, Agent Runtime, Agent SRE, Agent Compliance, Agent Lightning. Document architecture; identify which components map to current harness primitives and which are net-new.
2. OWASP Top 10 for Agentic Applications (December 2025 publication). Enumerate all 10 risks; map each to G1-G10 of harness goals; classify per-risk current coverage as enforced/advisory/missing.
3. OPA Rego vs AWS Cedar policy-as-code: feature comparison for our use case (hook-based gates, Boolean exit gates, signing-aware policy). Recommendation with rationale.
4. OpenTelemetry GenAI semantic conventions (`gen_ai.*`): current adoption in `scripts/global/event-schema-v3.js`; gap analysis to full adoption.
5. ISO/IEC 42001 Plan-Do-Check-Act structure: map our current enforcement catalog to PDCA cells; identify cells without coverage.
6. NIST AI Agent Standards Initiative (Feb 2026): summary of published standards and applicability to our identity/signing layer.
7. Self-attribution bias research (arxiv 2603.04582 and 2604.02174 plus follow-ons): confirm our deterministic-rubric (#1575) direction; identify additional mitigations beyond the rubric.
