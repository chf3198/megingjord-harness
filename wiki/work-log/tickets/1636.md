## Goal

Add a versioned, append-only \`decisions.md\` (or similar canonical path) per repo where every architectural decision is appended as a structured block. File-based, fully portable (no network or service dependency). Complements F2 Projects v2 (network-required, live) and F5 Discussions (network-required, decisional). Trade-off: latency for read-side is zero; write-side has merge-conflict risk.

## Portability annotation (per #1628)

- **Resource dependency**: NONE. Pure file in repo.
- **Network dependency**: NONE (after clone).
- **Classification**: PORTABLE BY DEFAULT. Works for every operator including air-gapped.

This is the **fallback decisional surface** for operators who don't have GitHub Discussions access (air-gapped, offline-during-decision-time).
