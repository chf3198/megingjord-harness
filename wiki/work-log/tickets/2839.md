## Problem
Four UX issues degrade first-time visitor experience:
1. Nav tabs wrap to 3 rows on viewports < ~900px
2. 5 icon-only action buttons have no text labels (accessibility violation)
3. Live Activity detail column text clips without ellipsis
4. Fleet Health uses full locale date; Live Activity uses time-only (inconsistent)

## Acceptance Criteria
- [ ] Nav: `overflow-x: auto; white-space: nowrap; scrollbar-width: thin` — all tabs on one scrollable row at 768px
- [ ] Each of 5 action buttons has `aria-label` + visible `<span class="btn-label">` text (Reset, Timer, Edit, Hint, Contrast)
- [ ] Activity detail cells: `overflow:hidden; text-overflow:ellipsis; white-space:nowrap; max-width:220px`
- [ ] `fmtTs(ts)` utility function added; all Fleet Health timestamps use it (time-only format)
- [ ] Screenshot evidence at 768px and 1440px attached to PR
- [ ] `npm run lint` passes on all modified files
