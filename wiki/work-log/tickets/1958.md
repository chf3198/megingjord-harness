## Body

Parent Epic: #1956

## Problem

When Team A authors a config file that Team B's runtime parses (the #1917 pattern), accountability for schema-correctness is currently ambiguous. The #1917 manager-handoff signed off on presence-of-references, not on whether the generated file would parse in the target runtime. No instruction file establishes that the target-runtime team owns the schema/contract test.

## Scope

Establish a cross-team artifact-write contract in the instruction set: when Team A writes Team B's runtime config, Team B owns the schema/contract test that must pass before the manager-handoff can be marked complete.
