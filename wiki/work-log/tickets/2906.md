## Body

## Problem
An agent can self-declare `lane:trivial` on any pull request diff regardless of actual size (Gap G-03). The lane determines which CI validators run — trivial skips several gates. There is no check that the diff size is consistent with the declared lane.

## Acceptance Criteria
- [ ] In `scripts/global/megalint/manager-handoff.js`, add a diff-size check: if declared lane is `lane:trivial` but diff exceeds N lines (suggest 50), emit a BLOCK with rationale
- [ ] Configurable threshold via `config/governance-rules.yaml`
- [ ] Add tests: `tests/lane-classification.spec.js`
