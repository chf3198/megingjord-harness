## Body

## Scope (Phase-1 child P1-5 of Epic #2029; adopted by Epic #2246)

`gen_ai.system` attribution from the wrapper per Epic #2029 AC2/AC4. `hamr-provider-wrapper.js`
(`emitStatSafe` → `appendCacheStat`) emits a `gen_ai.system` field on each `cache-stats.jsonl`
entry — derived from `effectiveProvider` — instead of relying on caller-side attribution.

**Provider→system mapping (remediated, Refs red-team #2232):** `gen_ai.system` MUST be a value in
`OTEL_GENAI_SYSTEMS` (`event-schema-otel-genai.js`). Several wrapper providers are NOT valid
semconv systems, so a mapping is required (not raw passthrough):
- Identity (already valid): `anthropic, openai, gemini, ollama, cohere, vertex_ai, bedrock`.
- Mapped to `other_system` (semconv catch-all): `openrouter, litellm, copilot, groq, cerebras`,
  and any unknown/unmapped provider.
- The raw `provider` field is PRESERVED on the event for fidelity; `gen_ai.system` is the
  semconv-normalized companion. (No conflict with `otel-gen-ai-content-validator.js` — that
  validates `gen_ai.*` emission events, a separate surface from `cache-stats.jsonl`.)

Refs Epic #2029
Refs Phase-0 source #2218

## Acceptance criteria

- **AC1**: `emitStatSafe` sets `gen_ai.system = providerToGenAiSystem(effectiveProvider)` on the
  `appendCacheStat` event; the raw `provider` field is retained.
- **AC2 (mapping)**: add `providerToGenAiSystem(provider)` (centralized in `event-schema-otel-genai.js`,
  alongside `OTEL_GENAI_SYSTEMS`): identity for valid systems; `other_system` for
  `openrouter/litellm/copilot/groq/cerebras` and any unknown provider. Return value is ALWAYS ∈ `OTEL_GENAI_SYSTEMS`.
- **AC3 (OTel compliance)**: the emitted `gen_ai.system` is always schema-valid (a test asserts
  `OTEL_GENAI_SYSTEMS.includes(value)` for every wrapper provider, incl. unknown).
- **AC4 (backward-compat)**: change is additive — no existing `cache-stats` field removed/renamed;
  existing JSONL readers (which ignore unknown keys) are unaffected. A test asserts prior fields persist.
- **AC5 (tests)**: ≥6 unit tests covering identity mapping (anthropic/openai/gemini/ollama),
  `other_system` fallback (groq/openrouter/litellm/copilot/cerebras), unknown-provider→`other_system`,
  schema-validity, additive-field preservation, and end-to-end emit via `emitStatSafe`.
- **AC6**: Lint clean.
