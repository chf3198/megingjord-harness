## Governance outbox + reconciler (offline fail-closed + auto-queue durability)
The durability layer so a chain link can never be SILENTLY skipped when GitHub/disk is unreachable.

## Acceptance criteria
- AC1: `~/.megingjord/governance-outbox.jsonl` — append-only; entries `state: queued|flushed` (never deleted); each carries `idempotency_key = sha256(canonical_json(event))`.
- AC2: `governance-outbox-flush` reconciler — `flock` lease (cron + SessionStart cannot double-flush); idempotent consumer (skip if the key marker already exists remotely); reconcile-failure emits a self-referential Tier-1 incident.
- AC3: `canonicalizeGovText()` — NFC + whitespace-collapse + trim + strip zero-width/control; enum fields stored as canonical tokens, not raw prose; hardens against injected/non-conforming fields (fail-closed).
- AC4: retention via existing `log-rotation.js`; writes via existing `log-redaction.js` (no new infra).
- test_strategy: tdd-pyramid + stress (concurrency, fault-injection, idempotency under partial failure).
