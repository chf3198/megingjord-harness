## Objective

Implement a deterministic lint gate that detects delegation phrasing in governed instruction surfaces and fails when phrasing implies user-executed steps.

## Scope

- Add/extend script under `scripts/global/` to scan:
  - `instructions/**/*.instructions.md`
  - `.github/copilot-instructions.md`
  - `AGENTS.md`
  - `CLAUDE.md`
  - `.codex/AGENTS.md`
- Flag phrases such as:
  - "you will need to"
  - "please manually"
  - "the user must"
  - "run this yourself"
- Allow explicit prohibition contexts (e.g., "Never ask the user...") to avoid false positives.

## Acceptance Criteria

- [ ] AC1: Script exits non-zero on true delegation phrasing in scoped files.
- [ ] AC2: Script ignores allowed prohibition contexts with tests for false-positive control.
- [ ] AC3: Output reports file + line + remediation hint.
- [ ] AC4: Script can run locally and in CI deterministically with no external calls.

## Constraints

- G3 zero-cost: local-only checks.
- G5 portability: no runtime-specific binaries required.
