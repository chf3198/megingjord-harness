## Background

The #1641 MCP migration audit ranked \`cross-team-lease.js\` as **priority #4** (High value). It runs the cross-team claim flow — correctness of lease creation and comment posting matters significantly for multi-team coordination. Schema validation via MCP reduces the risk of malformed lease comments silently failing validation.

## Current behavior

\`scripts/global/cross-team-lease.js\` uses \`gh issue comment\` and \`gh issue view --json\` via shell-exec calls.

## Target behavior

Replace with MCP equivalents:
- \`gh issue comment\` → \`mcp__github__create_issue_comment\`
- \`gh issue view --json\` → \`mcp__github__get_issue\`

Keep \`gh\` CLI as opt-out fallback path when \`MEGINGJORD_MCP_DISABLED=1\` is set (G5 Portability per #1629 contract).
