# #1454 — Lint Research-First Epic Phase Gate at close-time (block close if AC-R children incomplete)

> **Source**: github:issue/1454 | **state: CLOSED** | **Labels**: type:bug, status:done, priority:P2, area:governance, lane:code-change
> Mirror of `gh issue view 1454` (derived; edit the GitHub item, not this page).

## Body

## Problem

#1397 codified the Research-First Epic Phase Gate Rule. #1399 (open) is the planned docs follow-up to put it in `instructions/epic-governance.instructions.md`. Neither adds a LINT/enforcement gate.

A research-first Epic (filed with AC-R1..AC-Rn body markers) can today receive a CONSULTANT_CLOSEOUT and close attempt without any Phase-0 R&D children being filed or closed. The rule lives in instructions only — no automation.

## Observed evidence

Epic #1436 declares "Phase 0 — research-only per #1397 Phase Gate Rule" in its body and lists AC-R1..AC-R5. Within minutes of filing, a phantom CONSULTANT_CLOSEOUT was posted. Zero Phase-0 R&D children existed. The rule wasn't enforced.

## Goal

Add a lint check: if an Epic body declares Phase Gate compliance (AC-R* markers OR explicit "research-first" / "Phase 0" / "Phase Gate" language), block close until ALL R-children are CLOSED with Consultant rubric ≥7.

## Acceptance Criteria

- [x] AC1: Add `checkPhaseGateCompliance(epicBody, children)` to `scripts/global/megalint/epic-ac-traceability.js` (or sibling). Detects Phase Gate Epics via AC-R* checkboxes OR "Phase Gate" / "research-first" phrasing.
- [x] AC2: Resolves child issues via timeline cross-reference (same pattern as `.github/workflows/epic-traceability-lint.yml`).
- [x] AC3: For each AC-R1..AC-Rn in the Epic body, verify at least one CLOSED child references it.
- [x] AC4: Violation rule `phase-gate-rcildren-incomplete`; detail lists which AC-R markers lack closed children.
- [x] AC5: Wired into existing `epic-traceability-lint.yml` workflow + `closeout-schema` gate.
- [x] AC6: Test fixtures: Epic with AC-R* + 0 closed children → blocks. Epic with AC-R1..R3 + all 3 R-children closed → passes. Epic without Phase Gate language → skipped.
