# #2192 — Promote fleet-red-team from opt-in tool to hard-gate guardrail (research-first, ≥3 iterations)

## Body

## Goal

Convert Epic #2041's fleet adversarial red-team capability from an opt-in tool into a **hard guardrail enforced by code, not instruction**. Today nothing programmatically prevents an agent from skipping fleet-review at Consultant phase, bypassing HAMR via raw `curl`, or using the wrong tier and re-introducing the mis-recording defect P1-7 fixed.

Per goal-lens, G1 (Governance) is non-negotiable. Enforcement-via-instruction satisfies G3/G4/G7/G8/G9 but does NOT satisfy G1 strictly — a binding policy that relies on agent compliance is advisory, not governance.

## The enforcement gap (audit summary from Epic #2041 retrospective)

| Bypass vector | Current state | Real-world risk |
|---|---|---|
| Skip fleet-review at Consultant phase | No CI gate fails; no hook intercepts | Fleet review becomes optional under time pressure |
| Raw `curl http://fleet/...` instead of `dispatchRedTeam(...)` | Nothing prevents | HAMR cache-stats has no record; cost+observability opaque |
| Wrong tier passed to `wrapProviderCall` (e.g. `'fleet'` not `'fleet-local'`) | No validator | Re-introduces the mis-recording defect P1-7 fixed (provider mis-labeled as paid sticky-pick) |
| Fleet-review comment posted without classification block | No parser | Findings opaque; iteration loop breaks down |
| Codex + Copilot teams ship parallel wrappers that diverge from contract | No cross-team conformance check | Inter-runtime drift; per-team red-team patterns fork |

## Why a heavy multi-iteration Phase-0 is non-negotiable for this Epic

The previous Epic (#2041) closed Phase-0 in a single pass. That worked for additive plumbing but is too thin for a HARD-GATE Epic where:

- The wrong gate placement blocks legitimate work (false-positive merge failures)
- The wrong tier-vocabulary registration silently mis-records cost data (the very defect P1-7 fixed — and we should NOT trust a single Phase-0 pass to find an equivalent class of defect for THIS Epic)
- The wrong hook intercept point breaks operator workflows (canonical-main-enforcer-style overreach — see #2107 history)
- Cross-team enforcement contracts can't be retroactively patched once Codex + Copilot teams build against them

The Phase-0 child for this Epic is REQUIRED to converge across **≥3 iterations**, each with its own HAMR-routed cross-family fleet adversarial red-team review (dog-fooding Epic #2041's primitives — first real production consumer). Phase-1 child slate cannot be filed until Phase-0 converges with rubric ≥8 across G1-G9 AND no fleet finding rated ACCEPT remains open.
