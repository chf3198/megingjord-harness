## Problem
Four global functions called by `app.js` have no mock override in `transport-mocks.js`. This causes: Research Wiki empty, Ticket Log empty, Router Log empty, and Agent Baton state wiped every 5 seconds by `refreshAll()` calling the unmocked `getBatonState()`.

## Root Cause (baton staleness)
`refreshAll()` calls `pollEventBus()` (mocked → `[]`), so `evBaton=[]`. Falls through to `getBatonState()` (NOT mocked → `[]`), which OVERWRITES baton state written by `fixture-runner.applyEvent()`.
