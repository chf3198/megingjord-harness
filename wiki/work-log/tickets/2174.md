## Scope (Phase-0 research+planning child of Epic #2041)

## Research questions

1. **Prompt-template taxonomy** — which artifact types need distinct red-team prompts? (epic-scope, child-AC, COLLABORATOR_HANDOFF, ADMIN_HANDOFF, PR-diff, instruction-edit)
2. **HAMR routing** — exact `wrapProviderCall('ollama', ...)` invocation pattern; how to thread `tier='red-team'` for cost reporting separation; how to handle multi-host fleet (Tailscale fleet host vs local Chromebook)
3. **Finding-classification rubric** — how to formalize ACCEPT/REJECT/PARTIAL beyond ad-hoc judgment; iteration-convergence criteria (2-3 rounds per cross-family rule)
4. **Cross-runtime portability** — how do Codex + Copilot teams invoke the same red-team skill without coupling to Claude Code internals?
5. **Cost-floor protection** — when does fleet red-team add too much token cost; what's the agent-speed budget per cycle (<3K incremental tokens? <30s p95 per call?)
6. **Failure modes** — fleet host unreachable; fleet response malformed; fleet model hallucinated arxiv URLs (recurring per memory `feedback-red-team-iteration-pattern`); concurrent dispatch contention
7. **2026 industry research** — best practices in agent self-critique / red-team via secondary model; CAI ROOST agentic safety patterns; multi-model coordination patterns
