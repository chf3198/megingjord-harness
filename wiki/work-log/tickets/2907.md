## Body

## Problem
The collaborator self-check (`scripts/global/collaborator-checks.js`) currently runs as an advisory-only step. It is not enforced as a hard pre-PR gate. Rule 2.7 from the guardrail assessment is advisory (A tier). An agent can skip it and create a PR without running self-validation.

## Acceptance Criteria
- [ ] Add `collaborator-self-check` as a required step in `lefthook.yml` pre-push hooks (fail on non-zero exit)
- [ ] Ensure `collaborator-checks.js` exits 1 on any failed check
- [ ] Update the baton-gates CI workflow to require self-check evidence in COLLABORATOR_HANDOFF
- [ ] Add tests verifying the gate blocks a PR without self-check evidence
