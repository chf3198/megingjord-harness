## Objective

Re-create the Megingjord Harness Dashboard to meet an evolving set of requirements,
fully specified during Phase-0 research. The dashboard must serve a **Tri-Mode Architecture**:

1. **Local Mode** — a self-hosted web-app for operators passing data via localhost (SSE/REST).
2. **GitHub-Hosted Demo Mode** — a static web-app published via GitHub Pages that serves as an educational resource and live interactive demo (100% offline-capable, driven by static mock fixtures).
3. **VS Code Webview Mode** — preparation for Epic #2508; replacing global scripts with ES Modules so it can safely embed within strict IDE CSPs and postMessage transports.
