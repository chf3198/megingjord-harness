## Scope

Compute Goal Health Score (GHS) and emit it from `governance-audit.js` to `/tmp/governance-audit.json`. Bump `/quota` Worker schema to v4 with additive `goal_health_score_7d` field.

This is the foundational AC2 implementation. AC3 (sensor wiring) and AC4 (actuator wiring) are deferred to follow-on children.

## Implementation

1. New module `scripts/global/goal-health-score.js`:
   - Pure function `computeGHS({sensorValues, weights})` → `{score, contributing, weights_used, stale}`
   - Renormalization rule when sensors are null (per Phase-0 §2)
   - Floor at `Σ w_active ≥ 0.5` triggers `stale: true`
   - Default weights from Phase-0 §2 table (governance-audit 0.25, label-lint 0.15, Consultant-flags 0.20, PR-mentions 0.10, reopened-priority 0.20, operator-override 0.10)
2. `governance-audit.js` calls module with placeholder sensor values (real wiring is AC3); emits `goal_health` block in JSON output.
3. `cloudflare/hamr/routes/quota.ts` schema v3 → v4: adds `goal_health_score_7d: number | null` and `goal_health_stale: boolean`.
