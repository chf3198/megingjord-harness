## Description
Develop the automated ticket cataloging ingestion script in `scripts/wiki/auto-catalog-ticket.js` (strictly under 100 lines) and corresponding tests in `tests/validate-wiki-catalog.spec.js` that:
1. Triggers on issue closeout verification checks.
2. Extracts rubric scores from the `CONSULTANT_CLOSEOUT` comments and filters for scores $\ge 8$.
3. Compiles the resolved issue objectives, diff highlights, and rubric telemetry into a `WIKI.md`-compliant YAML frontmatter document.
4. Invokes `scripts/wiki/ingest.js` to place the file inside `wiki/sources/issue-<number>.md` and update catalog logs.
