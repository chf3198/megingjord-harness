## Objective

Systematically resolve all 15 issues catalogued in #2826 (UAT findings from the GitHub Pages demo at https://chf3198.github.io/megingjord-harness/). The demo is the primary external-facing artifact for the harness; quality gaps here directly affect G4 (privacy) and G2 (quality) goals.

## Problem Statement

The public demo exposes a second privacy leak in the Fleet/Devices panel (real hardware specs after #2821 only fixed static topology files), has 7 functional broken panels (NaN values, TypeError flood, empty sections), and multiple UX/layout issues that degrade first-time visitor experience.

## Scope

**In scope:** All 15 findings in #2826 — privacy remediation, mock data completeness, JS error fixes, UX responsiveness, and cosmetic polish.

**Non-goals:** Changes to local/vscode modes; backend API changes; navigation restructuring beyond tab overflow fix.
