## Goal

Replace soft, agent-readable governance (memory notes, prompt instructions, advisory hooks) with deterministic, non-bypassable guardrails wherever the same rule has been violated and re-noted in operator memory more than once.

## Motivation

Across the last ~30 sessions on this harness, the same anti-patterns recur and accumulate `feedback-*` memory notes (28+ in the operator memory index alone) instead of being structurally eliminated. Recent recurring traps:

- Stop-hook role-state drift (PR merged, ticket closed, but `roles.admin=false` in state-store)
- Cross-artifact prose collision (artifact-name substring inside other artifacts trips validators)
- Background-task dispatch followed by silent turn-end (Pattern B in `feedback-bash-sleep-block-recovery`)
- Stub cross-team-lease creation when CLI args omitted
- Python `json.dump` ensure_ascii default escapes Unicode tripping prettier CI
- Local-main-out-of-sync-with-origin during merge-evidence verification

Each was caught and worked-around in the same session, but the FIX was always another memory note. The 2026 industry term for the alternative is "harness engineering" — design the environment around the model so misbehavior is **physically impossible**, not just discouraged.

## Scope

Multi-phase Epic spanning the **5-tier guardrail strength spectrum** (per the Phase-0 research):

- **T0** Capability sandbox (microVM / gVisor / WASM) — currently absent
- **T1** Constrained decoding (XGrammar / strict JSON-schema) — currently absent
- **T2** Pre-tool hooks + policy-as-code (OPA / Cedar / APort) — currently partial (pretool_guard exists; OPA not adopted)
- **T3** External judge / classifier (NeMo / Llama Guard / Bedrock GR / Constitutional Classifiers) — currently partial
- **T4** Prompt instruction + memory feedback — currently HEAVY use (the layer this Epic exists to demote)

Phase-0 research child enumerates the spectrum and recommends an additive evolution (no rewrite): adopt OPA at the tool-call boundary first (highest leverage), then XGrammar grammars for baton artifacts, then microVM sandbox for untrusted fleet/codegen execution.
