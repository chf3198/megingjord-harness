## Origin

## Design (consumes #2658 + #2659)
- `scripts/global/runtime-session-registry.js`: `registerSession(runtime, opts)` (upsert per session_id into `~/.megingjord/runtime-sessions.json`: runtime/session_id/cwd/pid/ts/expires_at); `activeSessions(registry, opts)` (pid-alive AND not-expired — self-cleaning, no SessionEnd hook needed); `attributeForeignWriter(currentRuntime, registry)` → the active runtime != current, or `undetermined` (honest; never guess).
- SessionStart hook `runtime_session_register.py`: registers the current runtime (via #2659 `detect-runtime.js`).
- Wire `canonical-main-wip-guard.js` (#2658): name the foreign writer in the advisory heads-up + quarantine manifest using `attributeForeignWriter`.
