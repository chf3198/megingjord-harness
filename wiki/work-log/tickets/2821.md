## Body

## Problem
The GitHub Pages demo at https://chf3198.github.io/megingjord-harness/ exposes real infrastructure details publicly:

- `context-flow.js`: real device names (`CB-2`, `CB-1`, `Win Laptop`), real hostnames (`penguin`, `penguin-1`) in tooltips, RAM specifics (2.7GB, 16GB), LiteLLM port (`:4000`)
- `llm-context.js`: `penguin-1`, `windows-laptop` device names
- `fixtures/scenario-1-healthy.jsonl` + `scenario-2-governance.jsonl`: real hostnames `36gbwinresource`, `penguin-1`, `openclaw`

## Fix
Replace all real hostnames, device names, RAM amounts, and port numbers with generic demo labels (`Dev-1`, `Dev-2`, `Fleet Host`, `fleet-win-01`, `dev-node-2`).

- [ ] No real hostnames in any publicly deployed file

- [ ] Context Flow diagram uses generic device labels
