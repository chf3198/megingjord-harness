Build the parity-validator as a new megalint sub-validator scripts/global/megalint/parity-validator.js. Consumes rule-cards from rule-card-extractor (P1.2). Implements rule-pair-equivalence comparison: identical-statement-and-enum (parity OK), disjoint-enum (class b/c drift), statement-contradiction (class a/e — look up carve-out registry), MUST-without-enforcer (class d — check pending-enforcement tag). Emits PR-time as megalint sub-validator + nightly cron via new workflow + on-demand via npm run gov:parity.
