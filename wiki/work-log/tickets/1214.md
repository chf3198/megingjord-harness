## Scope

Build `scripts/global/test-evidence-validator.js`: pure-function validator that the CI gate consumes.

## Contract

```js
function validate({ test_strategy, lane, comments, pr_files }) {
  // returns { ok: bool, reason: string, violations: [{rule, detail}] }
}
```

## Logic

1. If `test_strategy === 'none'`:
   - pass iff `lane` ∈ {trivial, docs-research, docs-only, research, config-only}
   - else fail: "test_strategy=none disallowed for lane=X; declare a strategy from matrix"
2. If `test_strategy === 'tdd-pyramid' | 'tdd-trophy'`:
   - pass iff at least one `tests/**/*.spec.js` file is in `pr_files`
   - else fail: "TDD strategy declared but no test file added/modified in PR"
3. If `test_strategy === 'contract-test'`:
   - pass iff PR contains either `cloudflare/**/test.ts` or schema-assertion comment in trail
4. If `test_strategy === 'golden-file'`:
   - pass iff `tests/fixtures/**` or `test-results/**` referenced in PR or trail
5. If `test_strategy === 'eval-harness'`:
   - pass iff `tests/eval/**` or eval fixture referenced
6. If `test_strategy === 'visual-regression'`:
   - pass iff `VISUAL_QA_EVIDENCE` block found in trail (per existing instruction)
7. If `test_strategy === 'drift-lint'`:
   - pass iff `docs-drift-maintenance` skill output cited in trail
8. If `test_strategy === 'peer-review'`:
   - pass iff CONSULTANT rating ≥7 cited in CONSULTANT_CLOSEOUT comment
9. If `test_strategy === 'manual-verify'`:
   - pass iff "manual verification" or "before/after value" block in trail
