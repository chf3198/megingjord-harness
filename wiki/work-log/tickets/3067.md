## Body

Phase-1 child of #3063 (Refs #3063). Phase-0 source: #3063 consensus research. Depends on #3065.

## Scope
Catch-all reconciliation that repairs anything a missed/failed event left stale; cannot silently no-op.

## Acceptance criteria
- AC1: daily cron (07:30 UTC) reconciles A from local git data (git show HEAD:path, no network) and B against gh truth, falling back to a checksum-validated cache when gh is unreachable.
- AC2: cache-use ladder — fresh cache (under 24h) logs Tier-1 advisory + records cache-age; cache over 24h AND gh unreachable escalates to Tier-2; both-paths-fail is always Tier-2 (never silent). Prolonged outage (over 48h) serves last-known-good static zip.
- AC3: emits schema-v3 audit events (repaired list, drift delta, and cache_used hits distinct from reconcile_error_rate) to dashboard/events.jsonl.
- AC4: publishes a static .zip snapshot of A/B/C to wiki/archive/static/ with an Ed25519-signed SHA256 manifest (Tier-0 artifact). Local fallback npm run wiki:reindex documented.
