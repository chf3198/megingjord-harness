## Body

## Problem
Agent decision pathways are not immutably logged (Gap G-19). When an agent makes a governance decision (approve/reject/escalate), there is no structured audit trail of the reasoning. This violates ASI08 (Insecure Communications — provenance of decisions) and EU Art.12/13 (record-keeping, transparency).

`incidents.jsonl` captures events but not structured decision context (inputs, reasoning, alternatives considered).

## Acceptance Criteria
- [ ] Create `hooks/scripts/baton_event_emitter.py` (or extend `tool_activity.py`): structured decision log entries with fields: `timestamp`, `role`, `decision_type`, `input_summary`, `verdict`, `rationale`, `ticket`
- [ ] Emit on: MANAGER_HANDOFF post, COLLABORATOR_HANDOFF post, ADMIN_HANDOFF post, CONSULTANT_CLOSEOUT post, any deny decision
- [ ] Append to `~/.megingjord/decisions.jsonl` (separate from incidents.jsonl)
- [ ] Log is append-only; no delete path
- [ ] Add tests: `tests/decision-logging.spec.js`
