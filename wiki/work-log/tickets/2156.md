## Why agent-speed-appropriate

## Deliverable

```yaml
# .github/workflows/wiki-lint-gate.yml
name: wiki-lint-gate
on:
  pull_request:
    paths:
      - 'wiki/**'
      - 'README.md'
      - 'vscode-extension/README.md'
permissions: read-all
jobs:
  wiki-lint:
    runs-on: ubuntu-latest
    timeout-minutes: 5
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: '20' }
      - run: npm ci
      - run: npm run wiki:lint
```
