# Global Governance Self-Anneal (2026)

## Summary

- Governance execution quality was strong in local ticket artifacts and validation evidence.
- Primary drift: local markdown baton choreography outpaced observable GitHub protocol evidence.
- Epic closure discipline needs a stricter re-scope-before-close rule.

## Key Decisions

1. Require a GitHub evidence block before consultant closeout.
2. Forbid post-hoc scope normalization at epic close.
3. Trigger workflow self-anneal when local-vs-GitHub governance divergence is detected.

## Web Corroboration Added

- PR-to-issue linking and auto-close behavior is default-branch scoped.
- Manual close action and close reasons are explicit lifecycle controls.
- Labels are canonical triage metadata and should remain part of closeout evidence.

2. Add governance verification harness/checklist (#142).
3. Track outcomes in future closeouts.
