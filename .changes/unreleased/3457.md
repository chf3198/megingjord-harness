- **WASM baton-FSM byte-identity gate (#3457, Epic #3411 carve-out):** promoted the WASM kernel rebuild-and-hash check to a blocking gate. `npm run fsm:wasm:check` (`build-wasm.js --check`) rebuilds `kernel.wasm` in-memory from `transitions.js` (pure-JS encoder, no external toolchain) and compares byte-for-byte against the committed binary, exiting non-zero on any drift. A required `fsm-wasm-integrity` CI job (widened path triggers) enforces it. Golden spec `tests/fsm-wasm-check.spec.js`.
