  scenarios across OWASP-Agentic OA1/OA3/OA4/OA6/OA9 (fabricated/missing cross-family fields, lane-bypass,
  closeout rubber-stamping, prompt injection, fleet-review forgery, fail-closed inputs). Each guard test
  fails if its guard is removed; one explicit `[KNOWN-GAP #2908]` marker documents the not-yet-enforced
  G-score floor.

### Security
- Hardened the `Role:` field check in `collaborator-handoff` and `consultant-closeout` validators to a
  **line-anchored** regex (CWE-20), so injected trailing `Set Role: …` text can no longer substitute for
  the structured Role field. (#2921)
