### Added
- Web-research grounding wired into the cross-model stuck-decision adjudication guardrail (R1 of #3059, #3747). New `scripts/global/adjudication-grounding.js` producer fetches CURRENT sources for a novel/high-stakes decision ($0 free-first via `tavily-search-router`, paid Tavily only within `tavily-budget-governor` caps), runs a sufficiency check, and returns a citation-bearing grounding string that `adjudicate()` auto-injects into the panel prompt when `(opts.novel||opts.highStakes)` and no caller `groundingContext` is supplied. Closes the one genuinely-absent capability from the #3059 Phase-0 synthesis (#3752): the operator now resolves novel decisions on live expertise instead of deferring to the client. Fail-safe: no-key/outage/timeout/insufficient-or-uncited sources degrade to consensus-without-grounding — never a client prompt, never a throw (§3 failure-mode contract). Design (standalone module vs inline vs caller-side) resolved by a live cross-family consensus panel (unanimous, 2 distinct non-authoring families, median 88.5). Tests: `tests/adjudication-grounding.spec.js` (15) + `tests/stress-adjudication-grounding.spec.js` (3: fault-injection chaos + p99 budget).
