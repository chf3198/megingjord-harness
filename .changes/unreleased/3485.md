- **Fleet Advisor observability (#3485):** new `scripts/global/fleet-advisor-observability.js` emits one schema-v3 `fleet-advisor.report` event to `dashboard/events.jsonl` carrying per-{host,model} tokens/sec, cold-load rate, VRAM-pressure, per-class metrics, free-cloud-fallback rate, and the headline **dollars-saved** metric. A thin Prometheus exporter re-publishes the three core metrics with host/model labels, a live `dashboard/js/fleet-advisor-panel.js` panel streams them over the existing SSE pipeline, and a CI throughput floor (`config/fleet-throughput-floors.json` + `checkThroughputFloor`) fails the build when tokens/sec per model per tier regresses below a quantified floor (G7). Part of the Fleet Advisor capability (Epic #3414, §7).
