### Canonical harness feature catalog (Epic #3411 T1.1 keystone)

Adds `inventory/harness-feature-catalog.json` — the single source of truth for every harness
feature/surface (19 layers, 223 feature rows derived from the Phase-0 #3412 deliverable) and its
per-orchestrator parity contract — plus its draft-2020-12 JSON Schema, a zero-dependency
loader/validator (`scripts/global/harness-feature-catalog.js`, `npm run catalog:check`) that derives
all counts programmatically and content-hashes the catalog, and a golden test. One-step onboarding
(T2) and the per-feature x per-orchestrator parity matrix (T3) derive from this catalog. It extends
the Epic-1701 `governance/README.md` governance-manifest chain (asserts the anchor exists) and does
not fork it.
