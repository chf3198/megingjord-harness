### Added
- **Fleet dispatch substrate** (#3803, Epic #3126) — makes the $0 cross-model consensus lane actually reachable and auditable:
  - `config/fleet-hosts.json` + `fleet-registry.js`: settings-driven multi-host fleet list (no hardcoded IP). The second Tailscale host — sole source of the non-Qwen `deepseek`/`granite` families — was previously unreachable (HTTP 404) and now serves dispatches, restoring local cross-family diversity.
  - `config/model-capability-registry.json`: per-model family/size/thinking-flag/tok-s/timeout. Fixes `qwen3:32b` returning empty content and timing out at 306s (sets `think:false`, sizes the budget from cold-load + tok/s).
  - `fleet-preflight.js`: reports rater reachability + key presence + family BEFORE a run (key presence only — no values).
  - `fleet-resource-optimizer.js`: goal-weighted optimal-resource selection — quality is an adequacy threshold, then `$0` beats paid; paid picks carry an explicit `escalation_reason`.
  - `consensus-honest-stop.js`: emits `consensus-max=<N> on <families>` and stops instead of iterating against a structurally unreachable gate.
  - `fleet-roi-telemetry.js`: schema-v3 per-run event proving the workflow is net-positive.
- Operator guide: `docs/howto/fleet-dispatch-substrate.md`.
