### Cross-model review → remediation guardrail — Layer-B operationalized (Epic #3251)

Completes Epic #3251's deterministic review→remediation guardrail on top of the shipped
`baton-back` core (#3257), reuse-first (batch #3258 + #3259 + #3260 + #3261):

- **Layer-B finding router** (`cross-model-finding-router.js`) turns machine-readable
  cross-model review findings into baton-back markers via the shipped deterministic router
  (no LLM); hardened against adversarial/malformed model output.
- **Review dispatch adapter** (`cross-model-review-dispatch.js`) + operationalized
  `post-merge-consultant-audit.yml` (replaces the prior inert placeholder); degrades to an
  advisory `programmatic-only` when no model is reachable (never hard-fails).
- **Graded disclosure** (`review-coverage-resolver.js`) resolves the `review-coverage:*`
  rung with a 300s-TTL cache and an honest programmatic-only floor + client-UAT warning.
- **Phase-exit wiring + ledger** (`baton-back-set.js`, `baton-back-ledger.js`) persist the
  `## BATON_BACK` timeline marker, the `baton-back:open` / `review-coverage:*` labels, and a
  metadata-only redacted per-ticket observability event (off the #3573-fragile monolith).
- New labels registered; enforced-behavior documented in `role-baton-routing.instructions.md`.
