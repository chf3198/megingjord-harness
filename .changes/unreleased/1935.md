### Deploy atomicity — per-runtime rollback markers (#1935)

Added `scripts/global/deploy-atomic.js` (and `npm run deploy:atomic`): a transaction wrapper that backs up each runtime home before overwrite, restores it on per-runtime deploy failure, and appends schema-v3 rollback markers to `~/.megingjord/deploy-audit.jsonl` (redacted). `DEPLOY_ATOMIC=1` upgrades a partial failure to all-or-none (every succeeded runtime rolled back). Closes the gap where a partial `deploy.sh` failure left runtimes inconsistent with no recovery beyond `git revert`. Composes the existing `deploy.sh` rather than rewriting it.
