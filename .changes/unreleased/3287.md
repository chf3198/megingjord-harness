### baton-fsm core (Baton Authority Plane W1a) — #3287

Add `scripts/global/baton-fsm/`: a pure, deterministic `evaluate(state, event, evidence) -> verdict`
module — the heart of the Epic #3284 Baton Authority Plane. Ships a JS reference oracle plus a
real zero-dependency `kernel.wasm` (hand-emitted binary; byte-identical decisions to the JS kernel),
Ed25519-signed verdicts appended to a hash-chained replay-protected event log, signed evidence
provenance (forged/spoofed evidence rejected before evaluation), a fail-closed normalization grammar,
and independent `fsm_version` / `grammar_version` semver fields. Encodes the #3051 worktree-merge
precondition and the #3251 Admin->Collaborator baton-back as transition guards.
