### P1-a: `@megingjord/core` package (Epic #2508 Phase-1)

First increment of packaging Megingjord as a VS Code / Cursor / Antigravity extension (Hybrid Option C).
Adds a pure, dependency-free `packages/megingjord-core/` package (no `vscode` import) with the versioned
`MegingjordCoreApi` contract (`index.d.ts`): the G1..G10 goal-lens (`rank`/`compare`) and a carve-out
classifier mirroring the four retained human touchpoints. Exposes governance decisions only — never
credentials (G4). The extension host (P1-b) will wrap this and re-expose it via `activate()`.
