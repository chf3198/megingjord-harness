### Added
- HAMR MCP adapter (`scripts/global/hamr-mcp-adapter.js`): stdio MCP server exposing
  five HAMR capabilities (`governance_bundle_fetch`, `review_run`, `bundle_fetch`, `quota`,
  `substrate_health`) as MCP tools with Ed25519 DPoP signing. Tier-graceful: when HAMR is
  unreachable, `governance_bundle_fetch` falls back to a local `governance-bundle.js` build
  rather than hard-failing. Registerable via `.github/copilot-mcp.json`. Refs #3043.
