### Governance — F6 asserted-vs-observed contradiction probes (#3430, Epic #3425 P1-c)

`scripts/global/asserted-vs-observed-probes.js`: the F6 detector for the #3424 class — a baton artifact
asserts a state; a cheap read-only probe falsifies it. The catalog covers `worktree_residual_risk`,
`commit`/`branch` existence, PR-merged, and the all-ACs-pass claim. The centerpiece is a **squash-aware
worktree probe**: it uses `git cherry` **content-equivalence** (not commit-reachability), so a
squash-merged branch is correctly treated as merged and never re-creates the #3424 false-positive.
Each probe is budgeted (≤750ms/probe, 3s total cap → `inconclusive`), confidence-tagged
(`high` pure-local = blocking-eligible; `medium` network = advisory-only), fail-open (a probe that can't
run yields `inconclusive` + a `probe_error` row, never a false contradiction), and redacted before
emission. Wired into `review-point-checkpoint.runCheckpoint` as F6 `probeCandidates`. Advisory.
