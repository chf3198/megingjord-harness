- **Client-prompt-rate observability + adjudication decision log** (#3405, Epic #3392 AC4): new `scripts/global/client-prompt-rate.js` makes the "minimize client interruption" goal measurable. `computeClientPromptRate()` reports non-carve-out client prompts / total decisions (the carve-out class is excluded; target → 0); `logAdjudication()` records every cross-model adjudication `{path, risk_tier, panel_scores, confidence, rollback_handle}` to the schema-v3 audit stream (G8); `reversibilityAudit()` records a rollback handle for an adjudicated state-mutation (G2/G6) and flags non-reversible ones. `makeGuardrailLogger()` returns a sink compatible with the #3401 adjudication-guardrail `logDecision` hook, so guardrail verdicts flow into the signal automatically. All writes are fail-open — observability never breaks the decision it observes.
