- **docs:health prose-doc health scan** (#3298): new `npm run docs:health` extends the
  #3068 wiki-health pattern to non-wiki PROSE surfaces (`docs/`, `instructions/`,
  `governance/`, root agent + community markdown). Flags **orphan** docs (nothing in the
  tree references them — invisible to the changed-PR-only doc-coverage gate) and **stale**
  docs (past their surface's `freshness_window`), emitting schema-v3 G8 events. Adds
  `config/doc-health-owners.yml` (the ownership + freshness half of the #2708 freshness
  policy, completing the owner map alongside the existing trigger matrix) and an advisory
  `docs-health-advisory` workflow. Prompt artifacts (skills/agents/.claude/commands) are
  deliberately excluded — their structural quality is governed by #3302.
