### Added

- `hooks/scripts/session_anomaly.py` (#2913): new module for real-time session
  behavioral anomaly detection (Gap G-15). Tracks aggregate session counters —
  `total_writes`, `total_reads_sensitive_paths`, `total_pushes` — and emits
  `ANOMALY_DETECTED` v3 events to `~/.megingjord/incidents.jsonl` when
  sequence-level thresholds are breached.
- `config/governance-rules.yaml` (#2913): `session_anomaly_thresholds` block
  with defaults `writes_in_session: 50`, `sensitive_path_reads: 10`,
  `pushes_in_session: 5` (configurable per-repo).
- `hooks/scripts/tool_activity.py` (#2913): calls `update_session_counters` on
  every tool invocation to maintain session-level aggregate state.
- `hooks/scripts/pretool_guard.py` (#2913): `enforce_session_anomaly` gate
  returns a human-review-required deny when any anomaly threshold is exceeded.
- `tests/hooks/test_session_anomaly_detection.py` (#2913): 9 Python unit tests
  covering threshold loading, counter updates, anomaly detection, incident
  emission, and pretool_guard integration.
- `tests/session-anomaly-detection.spec.js` (#2913): Playwright JS shim for
  test-evidence gate discoverability.

Closes gap G-15 (sequence-level behavioral anomaly detection absent).
Refs #2913, #2891 (Epic). OWASP ASI05 (Cascading Failures), EU AI Act Art.14.
