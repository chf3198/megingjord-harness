### Added

- `scripts/global/tier-assert.js` (Epic #2398 AC6): `assertTier(requiredTier, opts)` + `readMinimumTier()` — reads `MEGINGJORD_MINIMUM_TIER` and fails closed (returns a `fallback` action + operator-visible message) when a feature requires a higher resource tier than the operator asserts; advisory when the var is unset.
- `scripts/global/megalint/tier-tag-lint.js` (Epic #2398 AC3): validates resource-tier tags on `scripts/global/*.js` — untagged scripts are the tier-1 baseline; any script referencing a tier≥2 resource (HAMR/Cloudflare, Tailscale fleet, paid-provider keys) must carry a `// tier: N` tag ≥ the tier its resources imply. Plus `scripts/global/tier-tag-apply.js`, an idempotent tagger, and explicit `// tier: N` tags on the 68 above-baseline scripts.

### Changed

- `tests/orchestrator-compatibility.spec.js` (Epic #2398 AC10): adds a `tier-assert-minimum` surface verifying each orchestrator (claude-code/codex/copilot/antigravity) is correctly gated at the asserted `MEGINGJORD_MINIMUM_TIER` (tier-2 feature → fallback, tier-1 → allowed).
