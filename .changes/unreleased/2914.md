### Added
- Deploy-artifact integrity: `scripts/global/deploy-manifest.js` generates SHA-256 manifests of deployed runtime artifacts and `scripts/global/verify-deploy.js` verifies live artifacts against a stored manifest (exit 1 on mismatch). Hardened per cross-family review: optional HMAC signature over the manifest (`DEPLOY_MANIFEST_HMAC_KEY`) with fail-closed `REQUIRE_SIG`, symlink rejection (CWE-59) on manifest + target paths, read-once hashing to avoid TOCTOU (CWE-367), and argument validation before path construction (CWE-22). New `deploy:manifest:*` / `deploy:verify:*` npm scripts wire the hook points. (#2914)
