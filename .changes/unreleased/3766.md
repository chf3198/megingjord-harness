  a live `Stop` hook (`hooks/scripts/stuck_state_gate.py`) via a fail-safe Node bridge
  (`scripts/global/stuck-state-hook-bridge.js`). A detected behavioral stuck-state (tool-call loop,

  the cross-model adjudication guardrail **without a client prompt**; a genuinely irreversible/
  high-destructive gate still escalates via `human-carveout`. Advisory-only (emits a schema-v3 G8 event;
  never blocks the turn); blocking promotion remains deferred per the #3748 panel. Companion to the
  #3749 client-defer language guard — behavioral signals vs. explicit language.
