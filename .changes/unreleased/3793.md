### Added

- `scripts/global/mcp-adapter-gate.js` (#3793, Epic #3789) — single source of truth for the
  `MEGINGJORD_MCP_ADAPTER_ENABLED` opt-in flag (default-OFF, fail-closed). `mcpAdapterEnabled()`
  returns `{enabled, flag, advisory}` and emits a non-silent OFF advisory (G8) when the adapter is
  consulted while dark, suppressible via `MEGINGJORD_QUIET_RESOLVER=1`.
- New `opt-in-enabler` classification in `scripts/global/baton-bypass/env-flag-classifier.js`, with
  `MEGINGJORD_MCP_ADAPTER_ENABLED` registered (`ci_authority: false`). Resolves audit **C-8**:
  dark-merging the HAMR MCP adapter must not reuse the global `MEGINGJORD_HAMR_DISABLED` kill-switch,
  which would collaterally disable every HAMR Tier-2 subsystem.
- `tests/mcp-adapter-gate.spec.js` — 7 unit tests (default-OFF fail-closed matrix, enabled path,
  advisory emission + suppression, classifier registration). The `hamr-mcp-adapter.js` wiring lands
  with PR #3050 on child #3796; the gate ships and is tested here on `main` independently.
