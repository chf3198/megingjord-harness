### Added
- `scripts/global/prefix-cache.js` + `npm run prefix:cache-report` (#3140, Epic #3137 T2): `applyPrefixCache(request)` marks the stable system/instructions **prefix** of an Anthropic request with a `cache_control:{type:'ephemeral'}` breakpoint so paid calls prompt-prefix-cache the resident prefix (a large $-per-call cut on the non-free lane); `cacheCoverageReport(request)` reports whether the prefix is cache-eligible. Idempotent and **graceful** — a malformed request or cache miss is a normal uncached call, no behavior change. Deterministic, local, $0; reuses the existing `cache_read_tokens` telemetry for the savings signal.
