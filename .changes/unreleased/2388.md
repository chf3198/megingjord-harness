### Added

- `tests/orchestrator-compatibility.spec.js` — portable cross-orchestrator compatibility test suite. Runs via `ORCHESTRATOR=<name> npm run test:compatibility` (single-orchestrator mode) or `npm run test:compatibility` (parity-matrix mode for all 4 known runtimes). Emits parity report to `~/.megingjord/orchestrator-parity-<timestamp>.json`. Companion doc `docs/howto/orchestrator-compatibility-suite.md`. (#2388)
