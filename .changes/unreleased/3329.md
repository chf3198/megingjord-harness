### Added — AI-suggested acceptance criteria with reconciler measurability backstop (#1299 / #3329)

`scripts/global/ac-suggest.js` drafts 3–7 measurable Epic ACs from a problem statement (free
fleet/free-cloud LLM lane, G3; deterministic offline fallback, G6) and validates each through the
`epic-ac-reconcile.js` measurability backstop before offering it for HITL accept/edit/reject. The
backstop verifies *measurability* (computable from a known evidence source), not correctness-of-
intent (Microsoft 2026 — "no oracle for spec correctness"). Includes a replay-eval FP-rate harness
(`npm run ac-suggest:replay-eval`, <5% bar) and fixes a `closeout-rescope-parser.js` regex defect
where plain `- [ ] AC1:` ACs failed to parse (only bold `**AC1**` did).
