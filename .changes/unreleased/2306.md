## #2306 — parity-validator (Epic #2295 P1.6)

### Added

- `scripts/global/megalint/parity-validator.js`: Sentinel-pattern governance rule parity
  validator. Compares `config/governance-rules.yaml` (expected state) vs
  `rule-card-extractor` extracted cards (actual state). Implements 5 conflict classes
  (doc-vs-enforcement, enforcement-vs-enforcement, enum-drift, doc-vs-no-enforcement,
  authority-carve-out), carve-out suppression, MUST-without-enforcer backfill, and
  Sentinel gate decisions (pr_introduced + hard-mandatory → block; pre-existing → advisory).
- `scripts/global/megalint/index.js`: parity-validator registered in dispatcher.
- `.github/workflows/parity-validator-nightly.yml`: Daily 05:00 UTC cron + on-demand;
  opens `governance-rule-drift` issue on new conflicts with SHA-256 dedup.
- `npm run gov:parity`: on-demand parity check invocation.
- `tests/parity-validator.spec.js`: 17 tdd-pyramid+stress-test specs (all passing).
- `tests/fixtures/parity-validator/`: 12 fixtures covering full conflict-class taxonomy.
