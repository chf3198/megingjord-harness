### #3356 — Decouple worktree squash-merge probe from clean-only path

`worktree-inventory.js` now resolves merge state for dirty worktrees too (new `mergedToMainDirty`
signal for the #3358 reconciler), preserving clean-path `squashMerged` semantics and classification
ordering. Adds an `openPr` guard (absorbs #2691): a merged worktree with an open PR is never
classified removable. Probe-unreachable now emits a labeled degraded signal (conservative merged=false).
