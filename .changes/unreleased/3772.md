- **Secret redaction enforced on the wiki write-path (#3772, Epic #3719).** The wiki contract long *claimed*
  "log-redaction + validate-at-write" but the code never applied it — a credential in a mirrored ticket/PR body
  would be committed verbatim. Now `wiki-io.js#writePage` scrubs credential-class secrets (anthropic/openai/
  github-pat/aws/jwt/bearer) prevent-at-write (G8 signal on hits), and a new **required** `wiki-secret-scan`
  gate fails any PR whose changed wiki pages contain a credential-class secret. Email/IPv4 (PII) stay advisory
  to avoid corrupting legitimate technical wiki content.
