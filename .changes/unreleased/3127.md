### Added
- `npm run resident:budget` (`scripts/global/resident-budget.js`, #3127): a deterministic, $0 counter of the always-resident context set (instructions/*.md + CLAUDE.md + optional operator MEMORY.md) in bytes/estimated-tokens against a baseline (~59K) and target (≤30K), plus a MEMORY.md index pointer-length lint (≤200 chars). Advisory by default; `--strict` opts into a non-zero exit; `--json` for machine output. First Phase-1 deliverable of the tiered-memory architecture (Epic #3124); the live over-limit operator MEMORY.md was drained to pointers under the same ticket.
