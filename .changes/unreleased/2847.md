### Added

- Governed MCP tool adapters for fleet models (#2847, P1-0 child of #2802): a typed tool catalog
  (github_read, wiki_search, repo_map — read; github_self_comment — self-comment-write) behind an
  OWASP-OA2 authorization gate that default-denies unknown tools, not-allowed permission classes, and
  malformed args before any execution. GitHub calls are token-brokered (the operator `gh` token is
  never exposed to the fleet model, never in a tool result), and self-comments are guarded against
  impersonating a baton governance artifact — including invisible-character/NFKC evasion (OA9). New:
  `scripts/global/fleet-mcp-tools.js`, `scripts/global/fleet-mcp-broker.js`. Permission boundary set by
  client design direction (UAT 2026-06-09).
