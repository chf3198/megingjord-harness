## Added

- `scripts/global/ticket-activate.js` (#3045): Copilot-invokable CLI that writes
  the shared `active_ticket` governance state so Copilot BYOK sessions can
  unblock the pretool ticket-gate. Requires a real open GitHub issue and a
  ticket-shaped branch (`feat/N-slug` or `fix/N-slug`); refuses with an
  actionable message otherwise. Writes both session-keyed and nosession state
  file variants to prevent dual-file desync. Covered by 19 deterministic unit
  tests in `tests/ticket-activate.spec.js`.
