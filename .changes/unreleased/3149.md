### Added
- `scripts/global/benchmark-harness.js` (#3149, Epic #3147 S3): a thin workflow-benchmark harness. `runBenchmark({name, metric, baselineFile, budget, direction})` captures an improvable metric, loads/seeds a local **golden baseline**, emits a regression delta, and flags `regressed` when the metric worsens beyond budget (direction-aware: `lower-better` / `higher-better`); **seeds on first run (never fails)**. `p99(samples)` + `runConcurrent(fn, count)` are the load/chaos facet for state-mutating/concurrent surfaces. Deterministic-first; fleet only for LLM-in-the-loop metrics; premium never. The foundation the enforcement gate (#3150), the 3-gap backfill (#3151), and the repo audit (#3152) build on.
