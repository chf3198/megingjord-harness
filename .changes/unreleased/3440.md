### Runtime descriptors + schema (Epic #3411 T1.2)

Adds `inventory/runtime-descriptor.schema.json` (draft 2020-12) and one descriptor per orchestrator
under `inventory/runtimes/` (claude-code, copilot, codex, cursor, antigravity). Each descriptor is the
data-driven input from which the T2 `harness:add-runtime` scaffold emits every per-runtime artifact,
and each round-trips against the live registries (detect-runtime.js env markers, deploy.sh homes,
team-model-signatures.json substrate map) via `scripts/global/runtime-descriptor.js`
(`npm run runtime-descriptor:check`) + a contract test. Copilot carries the `ai-agent-value` deltaKind
(the #3041 detection gap); cursor and antigravity waive the `scripts` artifactClass with a promotionPath
to T2.3. Absorbs the former duplicate descriptor-schema child (T2.0).
