### Added
- **Governance decision engine (#2483):** a pure, IO-free policy-driven evaluator
  (`scripts/global/governance-decision-engine.js`) that deterministically resolves
  which check-set gates each baton role transition
  (`manager_to_collaborator` → `consultant_to_done`), per lane and runtime profile,
  from `config/governance-decision-policy.json`. Replaces discretionary per-validator
  gating with one deterministic evaluator — the mechanism for autonomous full-role
  workflow completion (Epic #3391/#3392 operator-autonomy contract). Ships **shadow-mode**:
  feature-flagged OFF (`MEGINGJORD_DECISION_ENGINE`), enforces nothing, deletes no existing
  validator. Fail-closed on malformed policy or missing check input. Semantics ratified by
  free cross-family model consensus (Meta+Mistral, 2026-07-13).
