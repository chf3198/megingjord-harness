### P1-a: hard-floor override enforcement (Epic #2892)

Ships the enforcement primitive for the per-repo guardrail-override contract: `config/override-hard-floor.json`
(the versioned Tier-H list of security/provenance controls that are NEVER overridable) and a fail-closed
`scripts/global/megalint/override-contract.js` validator that rejects any per-repo override naming a
hard-floor control (and rejects everything if the hard-floor config is unreadable). Net-positive tightening:
it makes self-serve weakening of security controls impossible; it adds no new bypass surface.
