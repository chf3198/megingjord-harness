Per-call timeout and below-diversity-floor panels degrade to a documented self-resolution fallback — never a client prompt, never a throw. Reuses the existing $0 `fleet-decision-oracle` + `free-cloud-dispatch` substrate; websearch grounding is gated to high-stakes/novel decisions only (G3). Anti-goal honored: a security-weakening option is routed to the human, never auto-executed.
