### Added
- Pre-push closeout-presence gate (#3028, C3 of Epic #3026, G1). New `scripts/global/open-pr-closeout-check.js` reuses #3025's shared `scanOpenPRs` to detect open PRs whose linked issue lacks a CONSULTANT_CLOSEOUT and warns at push time. Advisory-first: exit 0 + warning by default; `CLOSEOUT_CHECK_BLOCK=1` promotes it to a blocking (exit 1) gate; `[skip-closeout-check]` in the latest commit subject overrides. Wired into `lefthook.yml` pre-push and exposed as `npm run check:closeout`. No detection logic is duplicated — the gate imports the already-stress-tested scanner (G2/G10). Covered by `tests/open-pr-closeout-check.spec.js` + `tests/stress-open-pr-closeout-check.spec.js`, both in the quality-gates deterministic CI list.
