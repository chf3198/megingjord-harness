### Added
- Stakes-tiered review-model selector (#2931, Epic #2926 C5): `scripts/global/review-stakes-router.js` routes a code review to the cheapest *adequate* cross-family reviewer by stakes derived from changed paths/labels — **routine** (docs/tests/config/dashboard) → `qwen2.5-coder:32b` fleet ($0); **standard** (general code) → `gemini-flash` free-cloud ($0); **high-stakes** (auth/crypto/IAM/governance-gates/`pretool_guard`/deny-list, mirroring the #2798 protected-path surface) → `gemini-pro` premium only. Enforces the cross-family invariant (reviewer model family ≠ author family) with a per-tier alternate on collision, so high-stakes never routes to a $0 tier. G3 default (routine+standard are $0); G2 where it matters.
