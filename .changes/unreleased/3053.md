### Governance — Admin merge-before-handoff guardrail (Epic #3051)

`ADMIN_HANDOFF` on a `lane:code-change` ticket is now flagged (`admin-handoff-without-merge`)
when the linked PR is CI-green and Admin-rated ≥93 yet still unmerged — the Admin role must
merge before handing off. A genuinely RED required-CI PR is never flagged (no forced bypass),
and documented exceptions (merge-evidence-override, baseline-drift) are honored. Enforces the
no-client-arbitration principle (#2578 / AC8): the Admin owns the merge decision and never
routes merge-authorization to the client. Closes #3053 #3054 #3055 #3056 #3057 #3058.
