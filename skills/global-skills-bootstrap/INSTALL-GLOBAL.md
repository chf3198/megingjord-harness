# Install Globally: global-skills-bootstrap

## Path

Place this folder in your personal skills directory:

- `~/.copilot/skills/global-skills-bootstrap/SKILL.md`

## Quick use

For a new repository, invoke:

- `repo-path: /absolute/path/to/repo`
- `mode: init`
- `profile: strict`

Recommended first prompt:

- "Include all global-skills in your initialization process."

The initialization will scaffold repo-level instruction routing, governance presence checks, and optional local hooks.
