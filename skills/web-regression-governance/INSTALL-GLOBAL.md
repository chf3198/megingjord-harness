# Install Globally: web-regression-governance

## Path

Place this folder in your personal skills directory:

- `~/.copilot/skills/web-regression-governance/SKILL.md`
- (optional compatibility) `~/.claude/skills/web-regression-governance/SKILL.md`

## Recommended invocation

Run after standards routing for website/static or web-app repositories:

- `app-type: auto-detect`
- `strictness: strict`
- `external-audits: required`

## Typical skill order

1. `repo-standards-router`
2. `web-regression-governance`
3. `github-ops-tree-router` (if required checks/rulesets need admin governance)
