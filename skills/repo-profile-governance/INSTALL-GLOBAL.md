# Install Globally: `repo-profile-governance`

Install this skill into your personal Copilot skills directory so it is available across all repositories.

## Recommended (Copilot)

1. Create the folder:
   - `~/.copilot/skills/repo-profile-governance/`
2. Copy `SKILL.md` into that folder.
3. Restart or reload your Copilot-enabled client if needed.

## Compatible path (other clients)

- `~/.claude/skills/repo-profile-governance/SKILL.md`

## Verify

Invoke with a short prompt such as:

- "Run `repo-profile-governance` mode=profile-audit scope=repo visibility=public"

Confirm you receive a `PROFILE_GOVERNANCE_REPORT` response block.
