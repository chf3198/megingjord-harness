# Install Globally: repo-standards-router

## Path

Place this folder in your personal skills directory:

- `~/.copilot/skills/repo-standards-router/SKILL.md`
- (optional compatibility) `~/.claude/skills/repo-standards-router/SKILL.md`

## Recommended use

Invoke at session start for a repo-level standards routing pass:

- `primary-type: auto-detect`
- `policy-profile: standard`

Then run the listed downstream skills in the returned `skills_to_run_next` order.
